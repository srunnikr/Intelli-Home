Intelli-Home Systems

Mission control dashboard gives you a succinct overview of the system status
We will have 4 distict sections on the mission control webpage as follows

1. TEMPERATURE : It shows the current ambient temperature of the room
2. HUMIDITY : It displays the humidity content of the room
3. LAMP STATUS : Shows the ON/OFF status of the lamp
4. DISTANCE SENSOR : Shows if any door is opened or not

If you any doubts please contact the customer care.
